# 资源声明系统

## 职责边界

- `scripts/prefabasset.lua` 只负责 DST 启动时的资源声明：`PrefabFiles`、`Assets`、`AddMinimapAtlas`。不要在这里做运行时外观逻辑。
- `scripts/musha/utils/anim.lua` 负责运行时动画外观工具：角色 `SetBuild`、延迟切换 build、手持物 `OverrideSymbol`、`AddOverrideBuild`。
- `modmain.lua` 和 `scripts/prefabs/musha.lua` 不再直接调用核心角色的 `SetBuild/AddOverrideBuild/OverrideSymbol`，统一通过 `MushaAnim` 工具层，避免外观切换、睡眠恢复、冰锤符号覆盖各写一套。
- 装备 prefab、普通物品 prefab 和状态图中仍可保留本地 `AnimState:SetBank/SetBuild/PlayAnimation` 初始化；这些属于实体自身资源初始化，不参与 Musha 角色外观状态机。
