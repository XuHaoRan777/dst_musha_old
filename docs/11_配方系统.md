# 配方系统

> 文件：`scripts/musha_adds_recipe.lua`

## 概述

所有 Musha 配方注册在自定义合成标签 `MUSHA` 下（`RECIPETABS.MUSHA`，图标 `mushatab.tex`）。  
配方根据 `difficultrecipe` 配置分为 **Easy** 和 **Normal** 两套，Normal 模式材料数量约为 Easy 的 1.5~2 倍。  
所有配方均设置 `builder_tag = "musha"`，仅 Musha 角色可制作。

## 智慧加成（Smart 配置）

通过 `smartmusha` 配置，在 `AddPrefabPostInit` 中修改 Musha 的科技加成：

| 配置值 | 效果 |
|--------|------|
| `verysmart` | `builder.science_bonus = 2`（相当于多 2 级科技） |
| `smart` | `builder.science_bonus = 1` |
| `normal` | `builder.science_bonus = 0`（无加成） |
