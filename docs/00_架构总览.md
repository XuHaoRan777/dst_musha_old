## 架构特征

### 优点
- 功能极其丰富：RPG 等级、4 种形态切换、3+ 伴侣、装备成长、独立 UI
- 多语言支持（中英韩俄）
- 丰富的配置选项（31 项可调参数）

### 问题
1. **核心 prefab 仍偏大** — `scripts/prefabs/musha.lua` 仍承载角色属性、等级、形态、资源、保存和部分战斗状态。
2. **Action/StateGraph 覆盖范围大** — `musha_adds_actions.lua` 和 `musha_adds_states.lua` 修改原版玩家行为链，影响范围需要每次重点回归。
3. **UI Hook 目录容易误解** — `data/screens/` 和 `data/widgets/controls` 通过自定义 `use` 加载 Hook，仍应按高风险 UI 兼容点处理。
4. **历史代码残留** — 存在注释旧逻辑、旧 prefab 文件和旧字段名，清理时必须优先保证存档兼容。
5. **第三方库老旧** — `libs/env.lua` 来自早期 Mod 加载方案，拆分模块时要额外注意严格环境下的全局变量访问。
