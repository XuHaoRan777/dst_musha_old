## 架构特征

### 优点
- 功能极其丰富：RPG 等级、4 种形态切换、3+ 伴侣、装备成长、独立 UI
- 多语言支持（中英韩俄）
- 丰富的配置选项（31 项可调参数）

### 问题
1. **modmain.lua 过度膨胀** — 已从 4673 行降到约 2968 行，但仍保留大量技能、UI Hook 和外观切换逻辑
2. **musha.lua 过度膨胀** — 6673 行单文件角色定义
3. **全局函数泄露** — `bodyguardwilson`, `exp_type_meat` 等未加 `local`
4. **UI Hook 位置混乱** — `data/screens/` 和 `data/widgets/controls` 通过自定义 `use` 加载 Hook，目录命名容易被误解为原版覆盖，仍建议迁移到专门的 UI hook 目录
5. **注释代码残留** — 多处 `--` 注释掉的旧逻辑
6. **文件名空格** — `prefabs/arong _old.lua` 包含空格
7. **第三方库老旧** — `libs/env.lua` 来自 2015 年
