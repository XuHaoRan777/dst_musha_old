# 本地化系统

> 文件：`scripts/speech_musha_*.lua`, `scripts/strings_musha_*.lua`, `modmain.lua`

## 概述

Musha mod 支持 4 种游戏内语言，由玩家在 Mod 配置第一项手动选择。

---

## 支持语言

| 语言 | Speech 文件 | Strings 文件 | 配置值 |
|------|-----------|-------------|--------|
| 英语 | `speech_musha_en.lua` | `strings_musha_en.lua` | `english` |
| 韩语 | `speech_musha_ko.lua` | `strings_musha_ko.lua` | `korean` |
| 中文 | `speech_musha_cn.lua` | `strings_musha_cn.lua` | `chinese` |
| 俄语 | `speech_musha_ru.lua` | `strings_musha_ru.lua` | `russian` |

---

## 语言选择

```lua
-- modlanguage 配置项可设为
"english"  → 英语（默认）
"korean"   → 强制韩语
"chinese"  → 强制中文
"russian"  → 强制俄语
```

旧配置残留的 `auto` 会按 `english` 兜底，不再扫描已安装语言包自动切换。游戏内语言只由 `modlanguage` 的手动选择决定。

## 文件结构

### Speech 文件 (speech_musha_*.lua)

角色检查/交互时的对话台词，挂载到 `STRINGS.CHARACTERS.MUSHA`：

| 类别 | 说明 |
|------|------|
| 行为失败 | BUILD / STORE / RUMMAGE / COOK 等失败提示 |
| 技能消息 | 主动/被动技能释放时的角色对话 |
| 战斗对话 | 攻击、受伤、击杀时的台词 |
| 宝藏猎人 | 嗅探/寻宝时的对话 |
| 音乐相关 | 演奏/充能相关文字 |
| 睡眠对话 | 入睡/唤醒/疲劳状态对话 |
| 表情/幽灵 | 死亡/鬼魂形态台词 |

### Strings 文件 (strings_musha_*.lua)

UI 显示文字，直接写入 `GLOBAL.STRINGS`：

| 类别 | 示例（英语） |
|------|------------|
| 角色名 | `STRINGS.NAMES.MUSHA = "Musha"` |
| 等级显示 | `STRINGS.MUSHA_LEVEL_LEVEL = "Level "` |
| 状态提示 | `STRINGS.MUSHA_TALK_SLEEP_GOOD = "[Musha can sleep well now]"` |
| 角色描述 | 其他角色描述 Musha 的台词（每角色一条） |
| 技能名称 | 各技能的名称和效果描述 |
| Badge 标签 | 法力/疲劳/耐力的 UI 显示名 |
| 音乐切换 | 音乐模式切换提示 |
| 死亡惩罚 | 死亡时的惩罚提示文字 |
| 伴侣对话 | Yamche/Arong 等伴侣的台词 |

### Strings 类别详细

| 字符串键前缀 | 用途 |
|-------------|------|
| `STRINGS.NAMES.*` | 所有 Musha 物品/生物名称 |
| `STRINGS.RECIPE_DESC.*` | 配方描述 |
| `STRINGS.CHARACTERS.GENERIC.DESCRIBE.*` | 其他角色检查 Musha 物品时的描述 |
| `STRINGS.CHARACTERS.MUSHA.*` | Musha 检查物品时的描述 |
| `STRINGS.CHARACTER_DESCRIPTIONS.musha` | 角色选择界面描述 |
| `STRINGS.MUSHA_*` | 自定义 UI 文字（等级、技能、状态） |
| `STRINGS.CRITTER_*` | 宠物相关对话 |
| `STRINGS.ACTIONS.*` | 自定义 Action 文字 |

## 加载方式

在 `modmain.lua` 中根据检测结果加载对应文件：

```lua
-- 加载角色对话
STRINGS.CHARACTERS.MUSHA = require "speech_musha_en"  -- 或 ko/cn/ru

-- 加载 UI 文字
modimport("scripts/strings_musha_en.lua")  -- 或 ko/cn/ru
```

每种语言同时加载 speech 和 strings 两个文件，分别覆盖对话和 UI 文字。

---

## 问题与建议

1. **游戏内语言不动态切换** — 游戏内 strings/speech 由 `modlanguage` 手动选择，并在 mod 加载时确定。

2. **无 fallback 机制** — 如果某语言文件缺少某个 key，会显示 nil 而非英语兜底。

3. **字符串散布** — 部分字符串直接写在 `modmain.lua` 和各 prefab 文件中（如 `STRINGS.CRITTER_*`），未集中到 strings 文件。

4. **无动态切换** — 游戏内语言在 mod 加载时确定，配置修改后需要重新进入世界。
