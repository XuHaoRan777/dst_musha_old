## 蛋孵化系统

### 基础蛋 (Musha Egg)

| 属性 | 值 |
|------|------|
| 预制件 | `musha_egg` |
| 合成 | amulet + 7~15 redgem + 7~15 bluegem (TECH.NONE) |
| 孵化产物 | `musha_small` |

**孵化机制：**
- 需要适当温度（不能太热/太冷）
- 状态流转：`uncomfy`(温度异常) → `crack`(准备就绪) → `hatch`(孵化完成)
- 放在地上启动孵化，放入物品栏暂停
- 玩家靠近准备好的蛋触发孵化
- 内置地图揭示器（1 单位/1s）

**失败掉落：**
- 过热 → loot_hot 掉落表
- 过冷 → loot_cold 掉落表

**孵化成功：** 生成 musha_small + green_apple_plant
