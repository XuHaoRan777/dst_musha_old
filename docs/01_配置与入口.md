# 配置项与入口系统

## modinfo.lua 维护方式

`modinfo.lua` 是 DST 运行时读取的单文件入口。为了保持游戏加载兼容性，发布态仍保留根目录 `modinfo.lua`。

配置源码已拆到 `modinfo_src/`：

| 路径 | 内容 |
|------|------|
| `modinfo_src/meta.lua` | Mod 基础元信息 |
| `modinfo_src/options/language.lua` | 语言配置 |
| `modinfo_src/options/keys.lua` | 快捷键配置 |
| `modinfo_src/options/difficulty.lua` | 难度配置 |
| `modinfo_src/options/recipes.lua` | 配方相关配置 |
| `modinfo_src/options/character.lua` | 角色机制配置 |
| `modinfo_src/options/visual.lua` | 外观配置 |
| `modinfo_src/options/containers.lua` | 容器配置 |
| `modinfo_src/options/misc.lua` | 其他杂项和历史注释配置 |

生成命令：

```powershell
.\tools\build_modinfo.ps1
```

检查命令：

```powershell
.\tools\build_modinfo.ps1 -Check
```

不要直接手改 `modinfo.lua` 的配置项；应修改 `modinfo_src/` 后重新生成。

## modinfo.lua 配置项总览（31+ 项）

### 语言设置

| 配置名 | 标签 | 默认值 | 选项 |
|--------|------|--------|------|
| `modlanguage` | Mod Language | `auto` | auto, english, korean, chinese, russian |

auto 模式通过检测已加载的语言包 mod 自动判断：
- 韩文包 → korean
- Chinese Language Pack / Chinese Plus → chinese
- Russian Language Pack → russian
- 否则 → english

## modmain.lua 入口结构

### 加载顺序

```
1. 读取全局引用 (GLOBAL.require, GLOBAL.STRINGS 等)
2. 读取所有 GetModConfigData (31项)
3. modimport("libs/env.lua")         ← 模块加载器
4. modimport("scripts/prefabasset.lua")   ← 资源声明
5. modimport("scripts/musha_adds_recipe.lua") ← 配方
6. 语言检测 + 加载语言文件
7. modimport("scripts/musha_adds_states.lua") ← 状态图
8. modimport("scripts/musha_adds_actions.lua") ← 行为
9. modimport("scripts/musha_adds_container.lua") ← 容器
10. 定义 40+ 函数 (combat, shield, sleep, music, visual 等)
11. 注册 35+ AddPrefabPostInit 修改原版预制件
12. modimport("scripts/mypower_musha_1.lua")    ← 法力状态
13. modimport("scripts/widgets/...")              ← UI 注入
14. modimport("scripts/difficulty_monster_dst.lua") ← 难度
15. use "libs/mod_env"(env)                       ← 环境设置
16. use "data/widgets/controls"                   ← 加载控件 Hook
17. use "data/screens/chatinputscreen"            ← 加载聊天输入 Hook
18. use "data/screens/consolescreen"              ← 加载控制台 Hook
19. 注册 TUNING.MUSHA.KEY* 快捷键
20. 注册 AddModRPCHandler (网络同步)
```
