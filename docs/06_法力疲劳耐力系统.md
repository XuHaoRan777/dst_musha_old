# 法力、疲劳与耐力系统

> 文件：`scripts/components/spellpower.lua` (251行)、`scripts/components/fatigue_sleep.lua` (152行)、`scripts/components/stamina_sleep.lua` (194行)

Musha 拥有三条独立的资源条，通过自定义组件实现，并通过 NetVar 同步到客户端 UI。

---

## 一、法力系统 (Spellpower)

### 属性

| 项目 | 值 |
|------|------|
| 上限 | 100 |
| 初始值 | 50 |
| 组件 | `spellpower` |

### API

```lua
self:GetPercent()       -- 获取百分比 (0-1)
self:SetPercent(p)      -- 设置百分比
self:DoDelta(delta)     -- 增减
self:DoRegen()          -- 自然恢复
```

变动时推送事件 `"spellpowerdelta"`。

### 恢复速率（每秒）

| 状态 | VeryEasy | Easy | Normal | Hard | Hardcore |
|------|---------|------|--------|------|----------|
| 深度睡眠 | +4.0 | +3.0 | +2.0 | +1.0 | +0.5 |
| 浅睡眠 | +1.0 | +0.5 | +0.25 | +0.1 | +0.1 |
| 女武神态 | +0.3 | +0.2 | +0.1 | +0.05 | +0.025 |
| 狂暴态 | +0.6 | +0.4 | +0.2 | +0.1 | +0.1 |
| 普通站立 | +0.1 | +0.05 | +0.01 | 0 | 0 |

### 异常消耗

中毒/冰冻/过热状态下法力以 -1.0/tick 流失，力场激活时归零不减。

### 法力等级阈值

| 等级 | 法力范围 | 用途 |
|------|---------|------|
| `vlow_spell` | < 15 | 无法使用大部分技能 |
| `low_spell` | 15 - 42 | 可用冰风暴 |
| `med_spell` | 42 - 73 | 中等技能可用 |
| `high_spell` | >= 73 | 所有技能可用 |

## 二、疲劳系统 (Fatigue Sleep)

### 属性

| 项目 | 值 |
|------|------|
| 上限 | 100 |
| 初始值 | 0（精力充沛） |
| 组件 | `fatigue_sleep` |
| 方向 | 值越高越疲劳 |

### 疲劳积累（每秒，活动中）

| 活动类型 | VeryEasy | Easy | Normal | Hard | Hardcore |
|---------|---------|------|--------|------|----------|
| 攻击 | +0.05 | +0.1 | +0.2 | +0.4 | +0.8 |
| 移动 | +0.01 | +0.025 | +0.05 | +0.1 | +0.2 |
| 劳动(砍/挖/钓) | +0.05 | +0.1 | +0.2 | +0.4 | +0.8 |

### 疲劳恢复（每秒）

| 条件 | 恢复速率 |
|------|---------|
| 光环开启 | -2.0 |
| 睡眠中 | -2.0 |
| 空闲(无光环) | 0 |

### 疲劳等级阈值

| 等级 | 疲劳值 | 效果 |
|------|--------|------|
| `vlow_tired` | < 15 | 无影响 |
| `low_tired` | 15 - 42 | 轻度疲劳 |
| `med_tired` | 42 - 73 | 中度疲劳 |
| `high_tired` | 73 - 100 | 严重疲劳 |
| `vhigh_tired` | >= 100 | 极度疲劳，产生 Debuff 特效 |

疲劳 Debuff 特效通过 `check_status_sleep` 周期任务（2s）生成 `sleep_debuff_30/50/70/90` 动画。

## 三、耐力系统 (Stamina Sleep)

### 属性

| 项目 | 值 |
|------|------|
| 上限 | 100 |
| 初始值 | 100（充分休息） |
| 组件 | `stamina_sleep` |
| 方向 | 值越高越精力充沛 |

### 耐力消耗（每秒，清醒状态）

| 时间段 | 温暖 | 消耗速率 |
|--------|------|---------|
| 白天 | — | 0 |
| 黄昏 | 否 | -0.02 |
| 黄昏 | 是 | 0 |
| 夜晚 | 否 | -0.05 |
| 夜晚 | 是 | -0.01 |
| 洞穴白天 | — | 0 |
| 洞穴黄昏 | 否 | -0.01 |
| 洞穴黄昏 | 是 | 0 |
| 洞穴夜晚 | 否 | -0.02 |
| 洞穴夜晚 | 是 | 0 |

### 耐力恢复（每秒，睡眠状态）

| 模式 | VeryEasy | Easy | Normal | Hard | Hardcore |
|------|---------|------|--------|------|----------|
| 深度睡眠 | +4.0 | +2.5 | +1.5 | +1.0 | +0.5 |
| 浅睡眠 | +2.0 | +1.5 | +1.0 | +0.5 | +0.25 |

光环模式额外 +0.5/tick。

### 耐力等级阈值

| 等级 | 耐力值 | 效果 |
|------|--------|------|
| `lowest_sleep` | < 5 | 极度虚弱 |
| `vlow_sleep` | 5 - 15 | 严重缺眠 |
| `low_sleep` | 15 - 42 | 缺眠 |
| `med_sleep` | 42 - 73 | 一般 |
| `high_sleep` | >= 73 | 精力充沛 |
| `full_sleep` | >= 95 | 完全休息 |

### 耐力→攻击速度修正

耐力直接影响攻击间隔：

| 耐力范围 | 攻击间隔(秒) |
|---------|-------------|
| >= 95 | 0.005 (极快) |
| 90-95 | 0.01 |
| 85-90 | 0.05 |
| 80-85 | 0.1 |
| ... | ... (逐级递增) |
| <= 0 | 0.9 (极慢) |

### 耐力→移动速度修正

耐力 < 30 时，移动速度降至 0.3-0.4x（战斗中除外）。

## 三条资源的交互关系

```
┌──────────┐     ┌──────────┐     ┌──────────┐
│ 法力(SP)  │     │ 耐力(STA)│     │ 疲劳(FAT)│
│ 0-100    │     │ 100→0   │     │ 0→100   │
│ 越高越好  │     │ 越高越好  │     │ 越低越好  │
├──────────┤     ├──────────┤     ├──────────┤
│技能消耗   │     │夜间消耗   │     │活动累积   │
│睡眠恢复   │     │睡眠恢复   │     │睡眠恢复   │
│被动缓回   │     │影响攻速   │     │影响debuff │
│异常流失   │     │影响移速   │     │          │
└──────────┘     └──────────┘     └──────────┘
       ↑                ↑                ↑
       └────── 睡眠是三条资源的共同恢复途径 ──────┘
```

## UI 显示

三条资源通过以下 Widget 在 HUD 上显示：

| 资源 | StatusDisplay 文件 | Badge 文件 |
|------|-------------------|-----------|
| 法力 | `spellpower_statusdisplays.lua` | `spellpowerbadge.lua` |
| 疲劳 | `fatigue_sleep_statusdisplays.lua` | `fatigue_sleepbadge.lua` |
| 耐力 | `stamina_sleep_statusdisplays.lua` | `stamina_sleepbadge.lua` |

每个 Badge 是一个圆形指示器，通过 `AddClassPostConstruct` 注入到 `StatusDisplays` 中。
