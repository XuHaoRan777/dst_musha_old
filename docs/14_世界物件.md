## 建筑

### 月树 / Dall (moontree_musha)

| 属性 | 值 |
|------|------|
| 预制件 | `moontree_musha` |
| AI | `dallbrain.lua` |
| 血量 | 800 + level × 2 |
| 掉落 | musha_egg, livinglog ×3 |
| 经验系统 | 有独立等级，击杀怪物获取经验 |

**经验加成（按怪物难度标签）：**

| 怪物难度标签 | 经验奖励 |
|-------------|---------|
| 无标签 | +1 |
| giant2x / monster2x | +1 |
| giant3x / monster3x | +2 |
| giant4x / monster4x | +3 |

**特殊功能：**
- 附近 Musha 装备可自动获得锻造效果（Forgelab 功能）
- 休息时周期生成 Dall 专属植物；跟随时从专属浆果丛/发光浆果召唤月龙（moonnutdrake）
- 季节动画变体（春/夏/秋/冬）
- 15+ 等级分档，逐级解锁更强属性

### 锻造台 (forge_musha)

| 属性 | 值 |
|------|------|
| 预制件 | `forge_musha` |
| 燃料上限 | 1000（初始 200） |
| 燃料分段 | 5 段，2 倍加成 |
| 容器 | 3×1（chest_yamche0 布局） |
| 光照 | 半径 1, 衰减 0.5, 强度 0.8 |
| 可砸 | HAMMER ×5 |

**功能：**
- 点燃后自动强化容器内的 `musha_items` 标签装备
- 每个燃料分段强化 1-5 级
- `cooked()` 函数处理容器关闭时的烹饪产物
- 5 级视觉效果随燃料等级变化
- Talker 组件播报当前燃料百分比

### Musha 帐篷 (tent_musha)

| 属性 | 值 |
|------|------|
| 预制件 | `tent_musha` |
| 耐久上限 | 100 |
| 睡眠消耗 | -0.2 / 2.5s |
| 待机消耗 | -0.01 / 2.5s |
| 燃料补充 | +20 /次（可燃物） |
| 光照 | 半径 5, 衰减 0.8, 强度 0.8 |
| 可砸 | HAMMER ×4 |

**睡眠效果（每 tick）：**
- 饥饿：-TUNING.SLEEP_HUNGER_PER_TICK
- 理智：+TUNING.SLEEP_SANITY_PER_TICK
- 生命：+TUNING.SLEEP_HEALTH_PER_TICK × 2（双倍回血）
- 体温：趋向 TUNING.SLEEP_TARGET_TEMP_TENT

**动画状态：** idle / place / hit / sleep_loop / destroy  
**外观切换：** tent_musha（关灯）→ tent_musha_on（开灯）→ tent_musha_broken（损坏）

### 烤箱 (musha_oven)

| 属性 | 值 |
|------|------|
| 预制件 | `musha_oven` |
| 燃料 | FIREPIT_FUEL_MAX × 5 段，5 倍加成 |
| 容器 | 3×3（chest_yamche5） |
| 可砸 | HAMMER ×4 |
| 标签 | campfire, structure, wildfireprotected |

**双模式系统：**

| 模式 | 切换 | 效果 | 火焰颜色 |
|------|------|------|---------|
| 加热 (on_fire) | TurnOff | 烹饪/烧制（log→charcoal, 可燃物→ash） | 红色 |

冷却模式自动添加 `fridge` 标签。

## 植物

### 月光灌木 (moonbush)

| 属性 | 值 |
|------|------|
| 预制件 | `moonbush` |
| 产物 | berries |
| 采集周期 | 随机 2-4 次 |
| 再生时间 | TUNING.BERRY_REGROW_TIME（递增） |
| 缩放 | 0.85 |
| 标签 | bush, moonbush, renewable, witherable |

**生长状态：**
- 90%+ → berriesmost 动画
- 33-90% → berriesmore 动画
- <33% → berries 动画

**交互：**
- 采集：减少 1 周期
- 当前未启用铲挖交互；没有 leader 时会短暂摇晃后消失
- 作祟：50% 概率摇晃，5s 冷却
- 需要 leader（follower 组件），否则消失

### 发光浆果 (moonlight_plant)

| 属性 | 值 |
|------|------|
| 预制件 | `moonlight_plant` |
| 产物 | wormlight_lesser |
| 标签 | moonlight |

**交互：**
- 可采集，按 `TUNING.WORMLIGHT_PLANT_REGROW_TIME` 再生。
- 需要 leader（follower 组件），否则短暂播放采集动画后消失。

`moonbush` 与 `moonlight_plant` 可被 Dall 标记为专属植物；Dall 跟随时只从这两类植物召唤 `moonnutdrake`，休息时月龙也只还原为这两类植物。

### 绿苹果植株 (green_apple_plant)

| 属性 | 值 |
|------|------|
| 预制件 | `green_apple_plant` |
| 产物 | green_fruit |
| 再生时间 | 8 天 |
| 缩放 | 随机 0.7-1.0 |
| 旋转 | 随机 0-360° |

**交互：**
- 采集：状态变为 picked
